# 당근게임 
- 제한시간안에 모든 당근을 클릭하는 게임 , 벌레 클릭시 실패

### 기술스택
HTML, CSS, JAVASCRIPT

- [ ] 일정 크기 이상 줄였을때 당근,벌레 이미지들이 화면 옆으로 넘어감
- [ ] 당근, 벌레 이미지에 마우스 올렸을때 크기 변경 안됨
